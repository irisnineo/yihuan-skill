# 玩家档案模板

用于记录长期偏好。用户本次输入优先级高于档案；档案只用于补足用户没有说明的默认值。

## 目标优先级

按用户长期偏好从高到低排序。

1. 角色图鉴。
2. 关键武器。
3. 版本限定活动。
4. 好感度投入。
5. 买房、载具、收藏等都市目标。
6. 方斯安全垫。

## 氪金倾向

- 付费档位：待填写。
- 礼包接受度：不买 / 只买高性价比 / 愿意按缺口买 / 预算上限。
- 双倍首充策略：保留 / 可兜底 / 优先使用 / 待询问。
- 角色抽卡礼包策略：不默认购买 / 黑于期望时补差 / 环石过低时用于抵扣抽卡环石消耗，保证活动混合支付后的方斯安全垫。
- 方斯礼包策略：不默认购买 / 只列高性价比 / 可按安全垫缺口购买。

## 角色抽取

- 角色目标：默认按期望规划 / 图鉴优先 / 命座优先 / 只抽强度 / 待询问。
- 武器目标：默认按期望规划，不轻易用环石和礼包抽取武器 / 不主动抽 / 只抽关键武器 / 待询问。
- 常驻资源：默认不计入限定角色或武器目标，除非用户明确允许。

## 都市消费

- 方斯安全垫：待填写。
- 猎人交易所：常规购买环石、三重钥匙和永恒的螺音 / 按缺口购买 / 暂不购买。
- 商店兑换策略：只换目标缺口 / 高性价比优先 / 暂不规划。
- 活动混合支付偏好：优先方斯 / 优先环石 / 保方斯 / 保环石 / 按性价比。

## 好感度投入

- 主培养：当前角色待填写；每日投入方案待填写。
- 其他角色：剩余送礼次数、礼物档位和单次方斯成本待填写。
