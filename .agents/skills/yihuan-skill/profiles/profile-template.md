# 玩家档案模板

本文件属于玩家档案层，用于记录长期偏好。用户本次输入优先级高于档案；档案只用于补足用户没有说明的默认值。

## 用户倾向

- 付费偏好：待填写，例如零氪、小月卡、大小月卡；是否接受高性价比礼包和首充双倍。
- 资源态度：待填写，例如保守留垫、均衡达成目标、激进追目标。
- 方斯态度：待填写，例如版本末保留 100-200 万方斯，买房版本和联动版本按本次目标单独判断。

## 目标优先级

1. 角色：待填写，例如全拿、只拿喜欢、强度优先、XP 优先。
2. 武器：待填写，例如每版本 0-1 把、每版本 1-2 把、只抽关键、用户指定才抽。
3. 版本活动：待填写，例如联动满档、限定活动满档、按性价比取档。
4. 好感度：待填写，例如固定日常投入、只养主推、资源紧张时后置。
5. 都市目标：待填写，例如买房、载具、收藏等大额消费。
6. 安全垫：待填写，例如版本末最低方斯和环石余量。

## 活动与商店

- 联动/限定活动：待填写。
- 猎人交易所：待填写。
