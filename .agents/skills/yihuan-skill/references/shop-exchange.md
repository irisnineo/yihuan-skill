# 异环商店兑换与体力活力规则

本文件属于游戏概念层，用于记录长期稳定的商店兑换比例、等效换算和购买优先级。版本限定商店、活动限购和临时兑换应放在对应版本参考文件中。

## 资料状态

- 标记为 `待你提供数据` 的项目只能在用户明确提供数值后补齐。

## 基础等效换算

```text
1 体力 = 1 都市活力 = 1000 方斯 = 1 环石
1 限定骰子 = 160 环石
1 三重钥匙 = 160 环石
```

等效换算用于比较机会成本；是否兑换仍要结合目标优先级、底仓和版本活动压力判断。

体力用于养成素材刷取；都市活力用于方斯获取，`1 都市活力 = 1000 方斯`。

## 商店刷新规则

| 商店/玩法 | 刷新规则 | 版本记录 |
| --- | --- | --- |
| 猎人交易所 | 每版本刷新；环石、三重钥匙、永恒的螺音限购固定 | 按版本开始日计入 |
| 集市兑换 | 每月刷新 | 跨月版本按刷新日分别计入 |
| 异境回收站 | 每月刷新 | 跨月版本按刷新日分别计入 |
| 勋章商店 | 不刷新 | 仅新增内容按版本事实计入 |
| 迷迭兑换 | 每月刷新 | 跨月版本按刷新日分别计入 |
