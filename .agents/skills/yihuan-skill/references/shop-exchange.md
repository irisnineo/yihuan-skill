# 异环商店兑换与体力活力规则

用于记录长期稳定的商店兑换比例、等效口径和购买优先级。版本限定商店、活动限购和临时兑换应放在对应版本参考文件中。

## 资料状态

- 本文件是规划用 reference，不从 `sources/` 自动生成。
- 标记为 `待你提供数据` 的项目只能在用户明确提供数值后补齐。
- 若只从截图或攻略材料推断，必须在修改本文件前获得用户明确指令。

## 基础等效口径

```text
1 体力 = 1 都市活力 = 1000 方斯 = 1 环石
1 限定骰子 = 160 环石
1 三重钥匙 = 160 环石
```

规划时不要把等效口径当成所有场景的无脑兑换建议。等效口径用于比较机会成本；是否兑换仍要结合目标优先级、底仓和版本活动压力判断。
