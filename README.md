# yihuan-skill

一个用于生成《异环》资源分析报告的 skill。

它会根据玩家库存、版本收入、角色武器目标、都市目标、商店兑换和活动档位，计算可用资源统计、目标消耗、联动支付路线和版本末余量，生成 HTML 报告。

## 查询方式

### 按日期查询

适合版本已经进行中，想查看某个日期时已开放与后续可获得的资源。

- 未指定日期时使用当前日期，也可以手动指定日期。
- 可用资源统计表显示：`当前 / 已开放 / 未开放 / 礼包 / 合计`。
- 报告会显示 `资源获取进度`。

### 整版本查询

适合版本开始前、还没开始玩，或想按整版资源做规划。

- 在输入里写 `整版本预估`、`版本开始前` 或 `从零开始规划`。
- 可用资源统计表显示：`当前 / v1.1 / 礼包 / 合计`。
- 不显示 `资源获取进度`。

## 使用

按日期查询：

```text
[$yihuan-skill](.agents/skills/yihuan-skill/SKILL.md)
我是nineo，版本 1.1，当前资源：方斯 900w，环石 19816，限定骰子 10，三重钥匙 25。
目标：角色全拿，武器 1 把，联动满，买房 750w。
```

整版本查询：

```text
[$yihuan-skill](.agents/skills/yihuan-skill/SKILL.md)
我是nineo，版本开始前，当前资源：方斯 900w，环石 19816，限定骰子 10，三重钥匙 25。
目标：角色全拿，武器 1 把，联动满，买房 750w，计划买 6r 和 30r 礼包。
```

报告生成到：

```text
reports/name-date.html
```

## 内容

- `reports/`：生成的 HTML 报告。
- `profiles/`：玩家档案，用于记录长期偏好和规划优先级。
- `references/`：已整理的版本资料、商店规则和通用计算规则。
- `sources/`：原始资料、截图和参考文档，仅作为备份。
- `.agents/skills/yihuan-skill/SKILL.md`：skill 的执行说明。

## 说明

这是非官方工具，数据和活动规则需要随游戏实装更新。资料来源于网络收集，仅供参考，请以游戏内实际内容为准。
