# yihuan-skill

一个用于生成《异环》资源分析报告的 skill。

它会根据玩家库存、版本收入、角色武器目标、都市目标、商店兑换和活动档位，计算可用资源统计、目标消耗、联动支付路线和版本末余量，生成 HTML 报告。

## 查询方式

### 当前版本预测

默认方式，适合版本进行中，查看当前库存到当前版本结束是否够用。

- 查询日期固定为生成当天。
- 可用资源统计表显示：`当前 / 未开放 / 礼包 / 合计`。
- 报告会显示 `资源获取进度`。

### 整版本预估

适合把当前资源当作指定版本期初库存，按整版收入做规划。

- 仅在输入里明确写 `整版本预估` 时触发。
- 可用资源统计表显示：`期初 / v<版本> / 礼包 / 合计`。
- 不显示 `资源获取进度`。

### 跨版本预估

适合当前版本进行中，但目标写到未来版本，例如当前 v1.1 时预测 v1.2 或 v1.3。

- 未来版本目标会自动触发跨版本预估，例如 `1.2 两个角色`。
- 缺少未来版本资料时，使用当前版本收益汇总估算对应版本收入，并在报告中标记为估算。
- 可用资源统计表显示：`当前 / v<当前版本>未开放 / v<中间版本> / v<目标版本> / 礼包x版本数 / 合计`。

## 查询示例

### 当前版本预测

```text
[$yihuan-skill](.agents/skills/yihuan-skill/SKILL.md)
当前资源：方斯 326w，环石 17028，限定骰子 5，三重钥匙 30。九百九十九夜活动未参与。
氪金预算：零氪。
目标：抽出伊洛伊，Draco 盲盒满。
```

### 整版本查询

```text
[$yihuan-skill](.agents/skills/yihuan-skill/SKILL.md)
我是nineo，1.1 整版本预估。
当前资源：方斯 900w，环石 19816，限定骰子 10，三重钥匙 25。
氪金预算：买 6r 方斯礼包。
目标：安魂曲1+1，卡厄斯，联动满，买房 200w。
```

### 跨版本预估

```text
[$yihuan-skill](.agents/skills/yihuan-skill/SKILL.md)
当前资源：方斯 500w，环石 18000，保时捷已抽 6。
氪金预算：大小月卡，大月卡等级奖励已领完。
目标：保时捷联动满，1.2 两个角色。
```

## 用户档案

默认使用 `profiles/nineo.md` 作为用户档案；用户本次输入优先于档案。

只有明确输入 `建立档案{name}`、`建档{name}` 时，才会基于模板创建 `profiles/<name>.md`；档案用于记录长期偏好和目标优先级。

## 内容组织

- `reports/`：生成的 HTML 报告。
- `profiles/`：用户档案。
- `references/`：已整理的通用计算规则、商店规则和版本资料。
- `sources/`：原始资料、截图和参考文档，仅作为备份。

## 说明

资料来源于网络收集，仅供参考，请以游戏内实际内容为准。
