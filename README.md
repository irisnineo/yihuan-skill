# yihuan-skill

一个用于生成《异环》资源分析报告的 skill。

它会根据玩家库存、版本收入、角色武器目标、都市目标、商店兑换和活动档位，计算可用资源统计、目标消耗、联动支付路线和版本末余量，生成 HTML 报告。

## 查询方式

### 按日期查询

适合版本进行中，查看生成当天已获取与后续可获取的资源。

- 可用资源统计表显示：`当前 / 未开放 / 礼包 / 合计`。
- 报告会显示 `资源获取进度`。

### 整版本查询

适合版本开始前、还没开始玩，或想按整版资源做规划。

- 在输入里写 `整版本预估`、`版本开始前` 或 `从零开始规划`。
- 可用资源统计表显示：`当前 / v1.1 / 礼包 / 合计`。
- 不显示 `资源获取进度`。

## 查询示例

### 按日期查询

```text
[$yihuan-skill](.agents/skills/yihuan-skill/SKILL.md)
当前资源：方斯 248w，环石 10353，限定骰子0，三重钥匙70。安魂曲已抽39，保时捷已抽8，猎人交易所未购买。
氪金预算：小月卡，不额外购买。
目标：抽出安魂曲，保时捷联动保底。
```

### 整版本查询

```text
[$yihuan-skill](.agents/skills/yihuan-skill/SKILL.md)
我是nineo，1.1 版本开始前。
当前资源：方斯 900w，环石 19816，限定骰子 10，三重钥匙 25。
氪金预算：买 6r 方斯礼包。
目标：2个角色，安魂曲专武，联动满，买房 200w。
```

## 用户档案

默认使用 `profiles/nineo.md` 作为用户档案；用户本次输入优先于档案。

只有明确输入 `建立档案{name}`、`建档{name}` 时，才会基于模板创建 `profiles/<name>.md`；档案用于记录长期偏好和目标优先级。

## 内容组织

- `reports/`：生成的 HTML 报告。
- `profiles/`：用户档案。
- `references/`：已整理的通用计算规则、商店规则和版本资料。
- `sources/`：原始资料、截图和参考文档，仅作为备份。

## 说明

资料来源于网络收集，仅供参考，请以游戏内实际内容为准。
